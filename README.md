# badapple-os64

64-bit thing similar to my previous project, `badapple-os`.

## Prerequisites

* A 64-bit capable UEFI computer with a decent CPU
* At least 200M of *usable* RAM.
* a 512M or larger flash drive

## Setup

1. Format your flash drive as FAT32.
2. Copy the contents of the zip file in the Releases page, such that EFI\ is in the root directory of the flash drive.
3. Insert it into your computer and boot off of it using the boot menu (Ensure Secure Boot is off)
